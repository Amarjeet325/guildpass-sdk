# Contributing to GuildPass SDK

First off, thank you for considering contributing to GuildPass SDK!

## Development Setup

1. Clone the repo
2. Install dependencies: `pnpm install`
3. Run tests: `pnpm test`
4. Start dev mode: `pnpm dev`

## Coding Standards

- Use TypeScript for all code.
- Follow the existing project structure.
- Write tests for new features.
- Run `pnpm lint` and `pnpm format` before submitting a PR.
- Keep the public API clean and well-documented.

## Pull Request Process

1. Create a new branch from `main`.
2. Make your changes.
3. Ensure all tests pass.
4. Update documentation if necessary.
5. Submit a PR with a clear description of the changes.

## Future Roadmap

Check the [README](./README.md) for the MVP limitations and future TODOs. We welcome contributions that help us reach those goals!
